# Hiring Fullstack Todo App

A fullstack TODO application built using the MERN stack (MongoDB, Express.js, React, Node.js).

## Overview

This TODO app was created following the recommended folder structure from the assignment documentation. The application uses React and TypeScript with Vite for the frontend, and Express.js with MongoDB for the backend.

## Project Structure

- **Frontend** - `client/` folder
- **Backend** - `server/` folder

## Prerequisites

- Git
- Node.js
- MongoDB

## Installation

Clone this repository using the following command:

```bash
git clone https://github.com/zafryimthisam/hiring-fullstack-todo.git
```

## Setup

For detailed setup instructions, please refer to:

- [Client Setup](./client/README.md) - Frontend setup and running instructions
- [Server Setup](./server/README.md) - Backend setup and running instructions

**Note:** You will need to open a terminal in the `client` folder to run the frontend and another terminal in the `server` folder to run the backend.
